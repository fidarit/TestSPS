# TestSPS

## Краткое описания задания
1. Написать по ТЗ SQL скрипты для создания БД
2. Создать WebApi сервис для возможности управления данными БД в таблице **Product**, используя EF Core
3. Создать Web-приложение на ASP .NET Core 7 MVC для управления данными об изделиях посредством сервиса, созданного в прошлом пункте

## Результат
1. см. папку [sql](sql)
2. см. проект [TestSPS.Api](TestSPS.Api)
3. см. проект [TestSPS.Web](TestSPS.Web)

## Сборка
1. Запустить Docker Desktop 
2. Ввести команду

```
docker-compose up
```
